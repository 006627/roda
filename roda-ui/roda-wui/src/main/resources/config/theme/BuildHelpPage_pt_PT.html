Adicionar tópico de ajuda nesta página

Para criar novos tópicos na página da ajuda, a única coisa necessária é adicionar o campo e a hiperligação na página Help.html:

    
<h2>Novo título (se necessário)</h2>
<ul>
    <li><a href="#theme/NovaPagina.html">Descrição do novo tópico</a></li>
</ul>

Depois disso, a página HTML do novo tópico deve ser criada na pasta de temas com o conteúdo pretendido.
