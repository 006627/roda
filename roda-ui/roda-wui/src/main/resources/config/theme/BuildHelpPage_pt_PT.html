Adicionar tópico de ajuda nesta página

Para criar novos tópicos na página da ajuda, a única coisa necessária é adicionar o campo e a hiperligação no ficheiro Help.html como no exemplo abaixo:

    
<h2>Novo título (se necessário)</h2>
<ul>
    <li><a href="#theme/NovaPagina.html">Descrição do novo tópico</a></li>
</ul>

A página de ajuda deve estar na pasta de temas [RODA_HOME]/config/theme/. Se não estiver, é necessário copiar o ficheiro HTML da ajuda que o RODA tem por omissão para a pasta e então editar.

Depois disso, a página HTML do novo tópico deve ser criada na mesma pasta de temas com o conteúdo pretendido.
