Build help page

To create new topics on the help page, the only thing required is add a new field and its links on the Help.html:

    
<h2>New title (if needed)</h2>
<ul>
    <li><a href="#theme/NewTopicPage.html">New topic description</a></li>
</ul>

After it, the new topic HTML page should be created on the theme folder with the desired content
