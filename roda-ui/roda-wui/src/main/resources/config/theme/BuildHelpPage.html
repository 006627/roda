Build help page

To create new topics on the help page, the only thing needed is adding a new field and its link on the Help.html file like the below example:

    
<h2>New title (if needed)</h2>
<ul>
    <li><a href="#theme/NewTopicPage.html">New topic description</a></li>
</ul>

The help page should be on the theme folder [RODA_HOME]/config/theme/. If not, it is necessary to copy the RODA default help HTML file to that folder and then edit.

After it, the new topic page should be created on same theme folder with the desired HTML content.
