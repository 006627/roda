Ajuda

RODA permite que especialistas em preservação se expressem através das funcionalidades fornecidas pelo software. O RODA foi projetado para ser fácil de usar e simples para pessoas não técnicas (por exemplo, utilizadores finais que não são conhecedores de arquivos ou tecnologia).

Mesmo que essas preocupações tenham sido sempre na mente dos progradores, às vezes há uma necessidade de descrever ou explicar como certas tarefas devem ser feitas no software.

Isto é o objectivo desta página. Aqui encontra-se informações úteis sobre as tarefas mais comuns que um utilizador pode fazer no software do repositório. A página de ajuda é organizada por secções, cada uma delas focando uma função especial do OAIS (Open Archival Information System).
