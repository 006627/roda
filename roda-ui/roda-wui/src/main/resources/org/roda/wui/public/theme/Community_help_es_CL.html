Comunidad de Ayuda

Hay dos listas de correo para ayudarle con Roda, uno para usuarios y otra para los desarrolladores.

Lista usuarios mailing

La lista de correo está dirigida a todo el mundo que utiliza RODA,como archiveros, productores o administradores del archivo que tiene algún problema o quiere discutir sobre el uso de características de Roda. Puede unirse a este correo Para hacer preguntas o participar en las discusiones acerca de todos los aspectos de la instalación, ejecución y el uso de RODA. Ver la lista de correo de noticias del sistema, conferencias de usuarios y anuncios desde el equipo de Roda.

Listado de mailing desarrolladores

La lista de correo de desarrolladores está dirigido a programadores que son (o le gustaría ser) parte del desarrollo de características RODA. Es el lugar para compartir ideas y discutir los cambios de código . Unirse a otros desarrolladores para dar forma a la evolución de la RODA software. La comunidad RODA depende de sus miembros para enmarcar requerimientos y arquitectura de alto nivel, y para facilitar la programación, las pruebas y la documentación del proyecto.
