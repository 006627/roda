Los registros de eventos son archivos especiales que registran eventos significativos que pasan en el repositorio. Por ejemplo, se lleve un registro cada vez que un usuario inicia una sesión, cuando la descarga se hace o cuando una modificación es hecha para un archivo de metadatos descriptivos. Cada vez que ocurren estos eventos, el repositorio registra la información necesaria en el registro de eventos para habilitarlo en el futuro la auditoría de la actividad del sistema. Para cada evento la siguiente información está grabada: fecha, los componentes involucrados, el método, la función, objetos de destino, el usuario que ejecuta la acción, la duración de acción, y la dirección IP del usuario que ejecuta el action. Usuarios son capaces de filtrar eventos por tipo, fecha y los demás atributos pueden ser seleccionados desde el panel del lado derecho.
