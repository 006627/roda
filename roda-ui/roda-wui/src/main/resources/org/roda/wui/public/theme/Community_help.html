Community help

There are two mailing lists to help you with RODA, one for users and another for developers.

Users mailing list

The users mailing list is intended for everyone that uses RODA, like archivists, producers or archive administrators that which to discuss problems on the usage of RODA features. Join this mailing list to ask questions or join discussions about all aspects of installing, running and using RODA. Watch the mailing list for news of software releases, user conferences and announcements from the RODA team.
